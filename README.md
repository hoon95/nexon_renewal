# CSS convention(Mozilla)

------------

> 1. display
> 2. list-style
> 3. position
> 4. float
> 5. clear
> 6. width / height
> 7. padding / margin
> 8. border / background
> 9. color / font
> 10. text-decoration
> 11. text-align / vertical-align
> 12. white-space
> 13. otehr text
> 14. content(내용) >> 레이아웃 >> 테두리/배경 >> 폰트 >> 기타
